## 2024-05-18 - AdminHome Component Memoization Optimization
**Learning:** In large React components like AdminHome with multiple lists and filters, array  and  calls executed inside the render function can cause significant performance degradation when typing in search fields or toggling simple states, due to O(N) operations running on every re-render.
**Action:** Use `React.useMemo` to memoize filtered list derivations and move static inline filters to stable memoized constants. This ensures O(N) list operations only occur when dependencies (like the search query, filter string, or underlying raw data array) change, preserving responsiveness for unrelated state updates.
