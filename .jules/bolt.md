## 2024-05-17 - React Component Optimizations\n**Learning:** In large list renderings like Admin panels, constant strings parsed repeatedly in `.filter` functions inside rendering paths can be a hidden drag. Using `useMemo` and hoisting out constants like `searchQuery.toLowerCase()` significantly avoids recalculations.\n**Action:** Always inspect array `.filter` and `.map` operations for repeated constant evaluations inside component render paths.
